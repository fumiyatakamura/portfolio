Description

This is a website that shows the revolution of music. A representative musician and a music device show up in each generation section. There is a button on each page to show the musician's name. This is controlled by the AJAX.

My Tasks

I worked as programmer in this project. I programmed AJAX call with pure JavaScript because of project requirement. And also, fixed musician background images are controlled by my JavaScript code.

Other information

We made this website before we learn database. This website has newsletter subscribe form, but it just validate e-mail address, and return a message. It actually doesn't work as subscription form.
